# Bayesian modeling and inference with Stan

The materials for a one-day class taught by @bob-carpenter and @mitzimorris.

## Bob Carpenter's slides

The top-level directory [`carpenter-slides`](/carpenter-slides)
contains Bob's slides.  You can make them in LaTeX as follows.

```sh
cd carpenter-slides
```

and then

```sh
pdflatex -output-directory=build slides.tex
```
